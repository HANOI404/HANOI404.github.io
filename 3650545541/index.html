JSON 简介

JSON 是一种数据格式, 不是一种编程语言, 而且是 JavaScript 的语法的子集.支持两种数据类型: 第一种是 JavaScript 中的基本类型(除了undefined外); 第二种是数组和对象(不支持函数).
